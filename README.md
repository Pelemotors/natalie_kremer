# דף נחיתה - ייעוץ ליזמים

דף נחיתה מקצועי בעברית עם עיצוב יוקרתי.

## מבנה הפרויקט

- `index.html` - המבנה הראשי של הדף
- `styles.css` - עיצוב יוקרתי עם צבעי שמנת, כחול כהה וזהב
- `README.md` - קובץ זה

## שימוש

פשוט פתח את `index.html` בדפדפן או העלה את הקבצים לשרת.

## תכונות

- עיצוב יוקרתי ונקי
- תמיכה מלאה ב-RTL (עברית)
- עיצוב רספונסיבי למובייל
- כפתור CTA עם קישור ישיר לטלפון
